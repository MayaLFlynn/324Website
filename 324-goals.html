Development

  • Medium-scale abstraction: use the principles and patterns of abstraction to design elegant, extensible, and maintainable software modules for medium-scale (multi-component) systems.
  • Medium-scale architecture: use common software architecture patterns to build medium-scale (multi-component) systems.
  • Domain-specific architecture: use knowledge of a particular computing domain to build medium- to large-scale systems within that domain.

Verification

  • Testing dimensions: employ fundamental testing strategies for designing software tests that adequately cover a medium-scale program.
  • Testing infrastructure: implement software tests that are fully integrated into a development pipeline, including development and deployment.
  • Advanced testing techniques: employ advanced testing techniques and technologies to improve software quality.

Design

  • Needfinding
  • Prototyping: develop (no-code) prototypes that address questions that arise during the design process.
    • Paper Prototype
  • Evaluation: design user studies to evaluate a candidate design against a variety of common usability dimensions.
  • User interface design: use the principles of interaction design to create effective user interfaces.
  • Accessible design: design software systems that account for a variety of user abilities and disabilities.

Engineering

  • Infrastructure: cultivate a development environment and set of tools that fosters effective software creation.
  • Self-learning: effectively learn on the job through resource curation, self-assessment, and reflection.
  • Process: employ common processes to effectively manage the software development lifecycle.
  • Collaboration: use best practices for communicating and collaborating with project teams and stakeholders.
    • Pull Request Collaboration
  • Professionalism: maintain integrity, reliability, and respect throughout the development process.
  • Quality: develop artifacts (both software and otherwise) that exhibit completeness, thoroughness, and polish.
